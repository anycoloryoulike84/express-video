# Node-Express

[ ](https://www.npmjs.com/package/github)

A simple node/express server.

## Installation

Install via npm.

```bash
$ npm init
$ npm install express nodemon redis bootstrap serve-static --save
$ nodemon app.js
```

or

Install via git clone (fork)
